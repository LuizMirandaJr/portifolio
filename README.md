Este projeto consiste em um site de portfólio pessoal estático, sendo a primeira versão e desenvolvido utilizando:

<h3>HTML5  |  CSS3</h3>

As fonts utilizadas foram importadas atraves de links do Google Fonts.
Para o formulário da seção de contato foi utilizado integração com o FormSubmit

O objetivo do site é apresentar informações pessoais e mostrar as tecnologias que tenho conhecimento. Também para mostrar
alguns projetos desenvolvidos com a utilização de outras tecnologias. Como o JavaScript.

O Site é composto da seguinte forma:

<h2>Cabeçalho</h2>
O cabeçalho foi criado utilizando 100% da altura da tela (100vh). Dessa forma, consegui deixar o menu junto a Logo como "capa" do projeto

Nessa seção conto um pouco de quem eu sou, minha profissão e as técnologias com que tenho algum conhecimento atualmente.

<h3>Projeto 1 - Descomplica Excel</h3>
Esse é o projeto de uma landing page para uma assessoria em excel. Projeto em fase inicial, focada em captar novos clientes para a empresa.
Esse projeto foi desenvolvido utilizando apenas HTML e CSS com @mediaquery para responsividade.
Conta com uma página extra de agradecimento após o envio do formulário de contato.

Esta disponível para acessar no github e com deploy na netlify para testes.

<h3>Projeto 2 - Calculadora de Combustível</h3>
Esse foi o meu primeiro projeto contando com JavaScript e totalmente idealizado por mim. 
Ele foi desenvolvido com HTML, CSS e JavaScript. A função do projeto é facilitar para as pessoas que tem dúvidas do quanto de combustível irão gastar em uma viagem.
No sistema, o usuário deve colocar a Distância que irá percorrer, a média de combustível que o veículo faz e o valor médio do combústivel, e através do JS, o sistema faz o calculo
de gasto em litros e o valor em reais do gasto.
O código foi bem simples, apenas utilizando constantes e eventos:

O projeto esta disponível para acessar no github e com deploy no github pages para testes.
- Github: https://github.com/LuizMirandaJr/calculadoraKM
- Deploy: https://luizmirandajr.github.io/calculadoraKM/

<h3>Projeto 3 - Calculadora de IMC</h3>
Projeto desenvolvido para um desafio em evento online.
Foco em ajudar as pessoas a saberem se estão dentro de um índice de massa corporal saúdável.
Foi desenvolvido em HTML, CSS e JavaScript. O sistema utiliza o peso e altura do usuário para calcular o IMC.
Para fazer o cálculo, utilizei condições como if e else if.

O projeto esta disponível para acessar no github e com deploy no github pages para testes.

Para o CSS, utilizei seletores aprendidos no curso e conhecimentos prévios de outros estudos.
Para melhorar a legibilidade do código, foi utilizada algumas técnicas como:
- Aninhamento de Seletores
- Propriedades Personalizadas (:root)
Além de formas de facilitar a responsividade, utilizando fonts com medidas em REM. 
Para fazer a responsividade foi utilizado @mediaquery com ponto de quebra em 768px.

Planejamento para a versão 1.01 onde incluirei o JavaScript com algumas funções. Uma delas por exemplo, será a possibilidade de alterar o tema do site entre Noite e Dia.
